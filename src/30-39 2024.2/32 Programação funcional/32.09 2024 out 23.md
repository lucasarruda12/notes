# List comprehension como aplicações de MAP e FILTER

```
[ t x | x ← xs ] ⇐≝⇒ map t xs
[ x   | x ← xs, p x ] ⇐≝⇒ filter p xs
[ t x | x ← xs, p x ] ⇐≝⇒ map t (filter p xs)
```
