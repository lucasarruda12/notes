## Match expression

```
match _ with
    pattern1 ~> exp1
    pattern2 ~> exp2
    ...
    patternN ~> expN

-- onde todos os expN tem o mesmo tipo α. E toda a expressão `match _ with` tem esse mesmo tipo α.
```

## Let

```haskell
let pattern = exp in ...
```

```haskell
let (q, r) = div(a, b) in ...
```

## Guards

O `Guard` é um guarda mesmo, que não te deixa ver a expressão se a condição não for verdadeira.

```haskell
foo x y 
    | cond1 = val1
    | cond2 = val2
    | otherwise = val

otherwise :: Bool
otherwise = True
```
